# Introduction
I built this tool to make it easy to run Linux containers on your Mac. It's based on the apple/container project, but I made some changes to improve the user experience.

## Get started
To install the tool, download the latest signed installer package from the GitHub release page and follow the instructions. Once installed, start the system service with `container system start`.

## Usage
You can use the tool to create and run Linux containers as lightweight virtual machines on your Mac. For example, you can pull and run images from any standard container registry, and push images that you build to those registries as well.

## What I changed
I added a new feature to allow users to easily manage their containers, and I also improved the performance of the tool by optimizing the underlying code. I also added some documentation to make it easier for users to get started.
